# Restaurant Analytics Dashboard

## Overview

The Restaurant Analytics Dashboard project analyzes restaurant data to identify patterns in sales, customer behavior, and order trends. The goal of this project is to convert raw data into meaningful insights using data analysis and visualization techniques.

---

## Dataset

The dataset contains **56,000+ records** including information such as:

* Restaurant details
* Orders and transactions
* Customer data
* Date and time information

---

## Tools & Technologies

* Power BI
* Power Query
* DAX

---

## Project Workflow

* Imported dataset into Power BI
* Inspected and cleaned data using Power Query
* Created a unique RestaurantID
* Built structured tables:

  * Restaurants
  * Orders
  * Customers
  * Date
* Established relationships between tables
* Created calculated columns and measures using DAX

---

## Key Insights

* Identified top-performing restaurants
* Analyzed customer ordering patterns
* Found peak sales periods
* Observed trends in customer activity

---

## Outcome

This project demonstrates:

* Data cleaning and transformation
* Data modeling and relationships
* DAX calculations
* Interactive dashboard creation
